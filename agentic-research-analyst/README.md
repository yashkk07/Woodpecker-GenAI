# Agentic AI Research & Business Insight Agent

This project implements an agentic AI system that autonomously analyzes long-form documents and extracts business-ready insights.

## Features
- PDF ingestion
- FAISS-based retrieval
- LangChain ReAct agent
- Local LLM via Ollama
- Tool-based reasoning
- Streamlit UI

## How it works
1. Upload a document
2. Provide a goal
3. Agent decides which tools to use
4. Outputs grounded insights

## Tech Stack
- LangChain
- Ollama (Mistral)
- FAISS
- SentenceTransformers
- Streamlit
